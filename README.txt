[ 1. 개요 ]

libGDX를 위한 도우미 라이브러리이다.

스크린, 월드, 유틸리티용 자료 구조, 화면 종속 타이머 등이 포함되어 있다.

자바와의 상호운용성을 최대한으로 고려하여 작성하고 자바 친화적 API도 별도로(이중으로) 제공하기 때문에 자바에서도 큰 위화감 없이 자연스럽게 사용 가능하다.



[ 2. 요구 사항 ]

32비트 운영체제 이상, 자바 JDK 1.8 이상



[ 3. 사용 예제 ]

https://github.com/Thyristor-Chopper/Endless-Dead 참고



[ 4. JAR 빌드 방법 ]

도스창에서 'gradlew assemble'를 실행하거나 Windows의 경우 build.bat 사용
